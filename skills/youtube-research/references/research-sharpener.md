# Research Sharpener

Sharpen a broad research question before generating search queries.

## When to skip

Skip when the question already points to specific results. The test: would two people reading this question search for the same thing?

## Process

1. Check context first. Read recent conversation history and any relevant project files (strategy/, docs/) before asking the user anything. If the answer is already there, use it.

2. Ask one question at a time. Each question must pass this test: would different answers lead to different search queries? Give 3-4 concrete options plus an open option. Stop when clear.

3. After clarification, generate 3-5 search queries that approach the topic from different angles. Vary synonyms, word order, and specificity.
